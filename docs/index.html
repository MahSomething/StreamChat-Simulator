How it works

Up and running in 3 steps

No technical knowledge required. Just Python, OBS and your content.

01
🐍

Start the server

Double-click iniciar-servidor.bat — the local server starts automatically and opens the controller in your browser.

02
🎮

Add to OBS

In OBS add a Browser Source pointing to http://localhost:8080/overlay.html. Enable transparency. Done.

03
▶

Hit Start

Load your users and messages in the controller, click Start — messages appear on your OBS overlay in real time.

AI Prompt

Generate messages instantly with any free AI

Copy this prompt, paste it into ChatGPT, Claude or Gemini — and get ready-to-import .txt files in seconds.

prompt.txt
Generate two text files for a Twitch-style live chat simulator. FILE 1 — users.txt Generate [30] users with names related to [your channel theme]. Required format — one per line: Name,#color FILE 2 — messages.txt Generate [300] chat messages related to [your channel theme]. - One message per line - In [English / Portuguese / French] - Topics: content appreciation, reactions, emojis, questions - Do not mention real people's names or specific locations Reply with both files separated as: --- users.txt --- (content) --- messages.txt --- (content)

Replace the highlighted values in yellow with your own details before sending to the AI.
